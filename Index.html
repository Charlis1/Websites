APPLICATION LAYER

Function: Closest to the end user, this layer interacts directly with software applications to provide network services. It enables functions such as file transfers, email, and web browsing. Protocols like HTTP, FTP, SMTP, and DNS operate at this layer.

PRESENTATION LAYER

Function: Translates data between the application layer and the lower layers, including data encryption, compression, and formatting. Example: Data encryption, JPEG, and PNG file formats.

SESSION LAYER

Function: Manages and controls the connections between computers (sessions), ensuring that data is synchronized and properly exchanged. Example: APIs and socket communication.

TRANSPORT LAYER

Function: Ensures reliable data transfer between systems with error recovery, flow control, and retransmission. Data Form: Segments.Example: TCP (Transmission Control Protocol), UDP (User Datagram Protocol).

NETWORK LAYER

Function: Determines the best path for data to travel between devices on different networks and handles packet forwarding, including routing through routers. Data Form: Packets.Example: IP addresses, routers.

DATA LINK LAYER

Function: Provides node-to-node data transfer and handles error detection and correction from the Physical Layer. Data Form: Frames. Example: MAC addresses, Ethernet

PHYSICAL LAYER

Function: Deals with the physical connection between devices, including cables, switches, and other hardware. Data Form: Raw bits (0s and 1s). Example: Ethernet cables, USB.
